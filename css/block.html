1) 기본 div (display: block)
block div 1
block div 2
block은 한 줄을 차지해서 다음 요소가 아래로 내려감.
2) div를 인라인처럼 (display: inline)
앞 텍스트
inline div A
inline div B
뒤 텍스트
inline은 줄바꿈 없이 옆으로 붙음. width/height는 거의 무시됨.
3) div를 인라인-블록처럼 (display: inline-block)
inline-block 1
inline-block 2
inline-block 3
inline-block은 옆으로 붙으면서도 width/height가 적용됨.
